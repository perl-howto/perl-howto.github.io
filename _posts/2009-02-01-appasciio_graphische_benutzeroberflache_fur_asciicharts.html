App::Asciio von Nadim Khemir, bietet über das mitgelieferte Programm asciio eine coole graphische Benutzeroberfläche (GUI) für ASCII-Charts.

Beispiel

	
	      .---.  .---. .---.  .---.    .---.  .---.
    OS API   '---'  '---' '---'  '---'    '---'  '---'
               |      |     |      |        |      |
               v      v     |      v        |      v
             .------------. | .-----------. |  .-----.
             | Filesystem | | | Scheduler | |  | MMU |
             '------------' | '-----------' |  '-----'
                    |       |      |        |
                    v       |      |        v
                 .----.     |      |    .---------.
                 | IO |<----'      |    | Network |
                 '----'            |    '---------'
                    |              |         |
                    v              v         v
             .---------------------------------------.
             |                  HAL                  |
             '---------------------------------------'

Augabeformate

Wird die mit App::Asciio erstellte Graphik in einer Datei mit der Endung .asciio gespeichert, kann die Datei später wieder mit App::Asciio bearbeitet werden. Die Dateiendung .pod exportiert die Datei ins POD-Format, .txt exportiert die Graphik als Plain-Text.

Summary

O-Ton Nadim: "I find myself starting it quite often just to draw a little diagram that helps me write better Perl code. Then I can keep it in the code itself."

Also, einfach mal ausprobieren. Es lohnt sich.
